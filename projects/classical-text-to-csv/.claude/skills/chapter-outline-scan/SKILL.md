# Chapter Outline Scan Skill

## Description

逐个扫描全文切块文档，提取候选章节标题、边界和结构线索。

## Purpose

先基于不超过 20000 字的切块文档识别全书结构轮廓，尽快产出可供下一步使用的章节候选信息，避免结构识别阶段直接处理整本书。

## Output

- `chapter_outline_candidates`：候选章节标题、边界、来源切块和线索列表

## Guidelines

1. 逐个阅读切块文档，先做局部结构判断，再汇总为全书候选线索，不要尝试重新通读整本书。
2. 优先寻找章节标题、卷号、序跋、空行分隔、文体切换等结构特征。
3. 每条候选结果都应带上来源切块、在切块内的位置，以及映射回全书的大致范围。
4. 输出应偏保守，宁可候选较粗，也不要长时间停顿等待完美结果。
5. 如果结构不明显，也要给出最粗粒度的候选边界，确保后续节点有可定稿的基础。
