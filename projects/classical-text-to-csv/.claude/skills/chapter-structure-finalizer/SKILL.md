# Chapter Structure Finalizer Skill

## Description

基于切块线索、切块清单和原始预处理文本，生成最终的结构化章节列表，并拆出按篇章保存的文档。

## Purpose

将前一步提取的粗粒度边界收敛为稳定、完整、可被后续批量节点直接使用的章节结构，并把全文正式拆成一篇章一个文档。

## Output

- `chapters_structure`：最终结构化章节列表
- `chapter_files_dir`：按篇章拆分后的文档目录

## Guidelines

1. 必须结合候选线索、切块清单和全文文本共同判断，不要脱离候选结果重新从零开始。
2. 先完成全书结构定稿，再将内容写入 `chapter_files_dir`，确保一篇章一个文档。
3. 输出时优先保证结构完整、边界连续和标题清晰。
4. 如果信息不足，应保守定稿，不要因为追求完美而长时间无输出。
